# Learning How To Code — Mini To-Do Addition

This workspace contained a simple static site. I added a small interactive To‑Do app:

- `index.html`: includes a `#todo` section and loads `script.js`.
- `style.css`: app styles appended for the To‑Do UI.
- `script.js`: add/remove/toggle tasks with `localStorage` persistence.

Run locally:

Open `index.html` in your browser, or run a quick web server:

```bash
# Python 3
python3 -m http.server 8000
# Then open http://localhost:8000 in your browser
```
